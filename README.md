# 🚗 Driver Drowsiness Detection System

A real-time AI system that detects driver drowsiness and micro-sleep using computer vision.

## ⚙️ Features
- Eye closure detection (EAR)
- Head drop detection
- Real-time alerts (alarm)
- Event logging
- Simple GUI

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- Tkinter

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
